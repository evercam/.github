## 📌 What is Evercam?

Evercam is a powerful visual monitoring and site analytics platform tailored for the construction industry. It unifies real-time video, time-lapse, drone imagery, 360° capture, and BIM (Building Information Modeling) overlays into a single interface to provide end-to-end project visibility.
